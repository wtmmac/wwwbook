# Mercurial的安装

## linux
### pip安装

	sudo pip install mercurial

### 源码安装
~~~
wget http://mercurial.selenic.com/release/mercurial-3.0.1.tar.gz
tar zxvf  mercurial-3.0.1.tar.gz
cd  mercurial-3.0.1
python setup.py build
python setup.py install
#验证是否已安装
hg --version
~~~
